# Simple Command Language for Text Editor
Proyecto de Teoría de la Computación – Implementación del Lenguaje “find / replace / delete”.

Este programa valida comandos simples siguiendo una gramática formal definida en el Proyecto I.

## Comandos válidos
- find:arg
- replace:old:new
- delete:number

## Tecnologías
- Python 3
- Lexer manual
- Parser descendente recursivo

## Ejecutar el validador
bash
python src/main.py
